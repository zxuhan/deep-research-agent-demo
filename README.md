# Deep Research Agent Demo

An AI agent that conducts deep research using Koog framework.

## Quick Start

1. Replace `YOUR_OPENAI_API_KEY` in `Main.kt` with your OpenAI API key
2. Modify the `query` variable to research any topic
3. Run the main function to start researching

## Project Structure

```
src/main/kotlin/
├── Main.kt                  # Entry point - configures and runs the agent
└── DeepResearchTools.kt     # Research tools: search, fetch, analyze, synthesize
```
